Responsive Design
The adaptive site in most cases is more expensive than the usual unadapped. This is explained by greater complexity of development.

First, for adaptive design, you need to create several layout options - under different sizes. In addition, additional design is required for each version. It takes more effort and web designer time. Consequently, the terms and cost of development increase.

Secondly, adaptive layout is more difficult. Implement it is a more laborious process. For this there are several ways, and the developer chooses the most appropriate, in his opinion, the decision.

Thirdly, the volume of testing increases. Before launching an Internet project into the world, it must be carefully tested. In the case of an adaptive site, all elements are tested sequentially on desktops, smartphones and tablets.
